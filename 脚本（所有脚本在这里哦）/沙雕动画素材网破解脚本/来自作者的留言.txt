如何使用？
1、在浏览器内下载油猴脚本这个插件（本人使用的Edge浏览器，也推荐使用Edge）
2、安装好油猴脚本后打开脚本内的实用工具（链接extension://iikmkjmpaadaobahmlepeloendndfphd/options.html#nav=utils）
3、点击里面的导入，先择ZIP文件
4、再点击导入即可导入成功

压缩包内一共存在4个破解脚本，分别对应的网站是：
1.1：橙皮素材破解脚本：https://www.chengpisc.com/svip
1.2：小鑫素材破解脚本：https://xiaoxinsc.com/svip
1.3：沈浪素材破解脚本：https://www.slsucai.com/
1.4：黑白素材破解脚本：https://www.hbsucaiwang.com/

橙皮素材破解脚本：
//---------------------------------------------------------------------------------------------
// 素材网：https://www.chengpisc.com/svip
// 这个脚本会在右下角增加一个按钮，很明显，点击后会进入登录状态（也就是假登录）
// 进入假登录状态后随便点一个素材就可以下载了
// 注意：不要点击下载图片按钮，点击下载图片按钮后假登录状态会失效，需要再次点击右下角的按钮
//---------------------------------------------------------------------------------------------
// 这个网页的制作者连复杂一点的都不会，登录状态随便改改就没了，这个素材网的破解脚本是这几个破解
// 脚本里面最小的。伪造一个登录状态就行。
// 原本我以为有多了不起呢
//---------------------------------------------------------------------------------------------

做这个的目的就是可以让更多的人用上优质的素材，破解脚本不会对任何素材网的服务器造成破坏，仅修改前端数据达到下载的目的。
奉劝有能力的人还是从正规途径购买素材，人家弄素材也不容易，也让人家回回本。
本破解脚本适用人群：穷的叮当响的人
本破解脚本不适用人群：0岁以下的人，1000岁以上的人，外星人，火星人，非人类，资本家，豪门子女，富一代，富二代，富三代，富N代，个人可使用资产达到1000以上的人群
